Au bac, j'avais pris la spécialité ISN (Informatique et Sciences du Numérique). Nous apprenions alors le langage Java en utilisant Javascool.

Connaissant déjà le Java depuis 3 ans à l'époque, je trouvais javascool extrêment limité : aucune classe possible (car tout le code est comme copié-collé dans la fonction main qui est d'ailleurs cachée), une limitation à 56ko de code, impossible de charger des images depuis le disque sans les afficher...

En décembre, on nous a demandé de faire une animation afin d'être évalué. L'enseignant s'attendait à voir une image qui bouge dans un décor.

Puisque je m'ennuyais, j'avais décidé de mettre un maximum de choses dans ces 56ko de code : définition des images dans le code lui-même (puisqu'on ne peut pas les charger depuis le disque), transparence, profondeur, effet de lumière, tilemap, collisions (la direction initiale du fantôme étant aléatoire)
