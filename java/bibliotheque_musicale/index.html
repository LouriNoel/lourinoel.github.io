Introduction

Il s'agit de l'implémentation en JEE d'un site de streaming musical en ligne. Cela a été l'occasion de déployer un serveur Apache Tomcat, d'utiliser des Servlets et des JSP (java server pages), et d'employer le framework Hibernate pour s'interfacer avec une base de données MySQL ou MariaDB.

L'équipe était composée de 3 personnes. Je me suis occupé en particulier du css (dont l'animation du passage de clefs de sol en 3D) et de l'internationalisation (support de différentes langues). Nous avons travaillé ensemble sur les autres aspects.
