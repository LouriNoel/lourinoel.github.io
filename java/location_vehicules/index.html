Introduction

Contexte : une application pour un employé d'une entreprise de location de véhicules (voiture, moto, avion). Un client l'appelle, l'employé remplit sa fiche et fait le lien avec le véhicule voulu. (vérification si le véhicule est disponible, etc)

Une application Java utilisant les composants Swing, dont le but était de bien apprendre à manipuler les layouts. Au final, comme mon binôme et moi-même connaissions déjà très bien cela, on en a profité pour faire des Generics, des Factory, des Streams, de la Serialization, de la recherche par tag et de l'auto-complétion, ...

Ça a donné plus de 100 classes et 6000 lignes de code.
